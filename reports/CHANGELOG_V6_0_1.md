# 电化学数据处理软件 V6.0.1 更新日志

发布日期：待定

## 版本定位

`V6.0.1` 为无激活重构版，目标是将原有电化学数据处理能力迁移到新的 `API + Web UI` 架构，并保留实验分析工作流。

## 本版重点

1. 移除激活限制
- 默认可用，不再依赖激活码机制

2. 重构专业工作台
- 默认进入专业模式
- 新增 AI 对话模式与项目管理模式
- 统一中文界面文案

3. 完整项目管理
- 支持项目创建、编辑、删除
- 支持历史查看、归档、删除
- 支持结果文件查看与打开
- 支持导出项目报告

4. LSV 对比分析增强
- 支持 LSV 对比表
- 支持叠加曲线图
- 支持指标柱状图
- 支持最近一次对比图回显
- 支持目标电流密度按历史真实结果自动识别

5. AI 模式增强
- 支持模型、API、超时等配置
- 支持提示词模板
- 支持会话重命名、删除、检索
- 支持 Markdown 回复显示
- 新增 v6 专用项目查询工具

6. 帮助系统增强
- 增加全局“使用说明”入口
- 支持中英文帮助文档
- 支持目录导航与章节跳转

## 详细更新

### UI 与交互

- 软件名称统一为 `电化学数据处理软件V6.0.1`
- 顶部导航固定为：
  - 专业模式
  - AI 对话模式
  - 项目管理
- 增加全局使用说明面板
- 参数区增加推荐值与说明提示
- 文件匹配方式与匹配值标签联动
- `iR` 补偿、目标电流密度等布局细节已优化

### 专业模式

- 支持 `LSV / CV / EIS / ECSA`
- 支持参数模板保存、加载、删除
- 支持绘图字体、字号、标题、坐标轴标题设置
- 支持电位手动偏移与 `RHE` 公式换算
- 支持实时显示当前换算偏移值
- 支持过电位开关与平衡电位输入

### 项目管理

- 增加项目列表与项目信息编辑
- 增加最近历史与历史详情
- 增加结果文件分组展示
- 支持复制路径、打开文件、打开目录
- 支持归档记录与显示已归档
- 支持 LSV 摘要、结果对比与项目报告导出

### LSV 对比

- 叠加曲线图用于查看原始曲线形态
- 指标柱状图用于比较 `E@j / η@j / Tafel`
- 目标电流密度不再手输，而是按项目真实历史自动识别
- 对比图可回显最近一次同类型结果

### AI 对话

- 会话支持新建、重命名、删除、检索
- AI 设置支持：
  - Provider
  - 模型
  - Base URL
  - API Key
  - 超时
- 提示词支持：
  - 电化学分析师
  - 结果总结模式
  - 论文写作模式
- 新增 v6 项目查询工具：
  - 当前项目摘要
  - 当前项目历史
  - 当前项目 LSV 对比候选数据

### 稳定性与测试

- 修复了 v6 历史写入与旧根目录历史混用问题
- 修复了 `run_id` 透传缺失导致结果文件无法回填的问题
- 修复了中文帮助文档乱码问题
- 修复了对比图接口和最近结果回显逻辑
- 增加了项目并发、对比图、AI 会话等回归测试

## 已知限制

1. AI 目前不是完整的项目级智能体
- 已支持部分项目查询能力
- 但不会自动读取整个工作台的全部状态

2. 柱状图指标依赖历史中已有对应结果
- 例如项目历史里只有 `E@10`
- 则无法直接比较 `E@50`

3. 当前发布形态仍偏向本地服务 + 浏览器
- 如需面向最终用户分发，建议后续增加桌面壳或 EXE 打包
